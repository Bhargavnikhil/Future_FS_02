# 🚀 LeadFlow CRM – Client Lead Management System

**LeadFlow CRM** is a modern web-based Client Lead Management System designed to help businesses manage leads, track client interactions, and monitor deal progress through a structured pipeline.

This project demonstrates frontend development skills, UI/UX design, and client-side data management using HTML, CSS, and JavaScript.

---

## 📌 Project Overview

LeadFlow CRM allows users to:

- Manage client leads in one centralized system
- Track lead status (New, Contacted, Converted)
- Add, edit, and delete lead information
- Store notes and follow-up details
- Search and filter leads easily
- View dashboard statistics
- Use a secure login system

The application uses **Local Storage** to simulate a database, making it lightweight and easy to run without backend setup.
